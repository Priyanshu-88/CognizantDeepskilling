# Java Practice & Handbook

This repository contains my Java practice programs and solutions created while learning the Java programming language.

The purpose of this repository is to document concepts, practice coding regularly, and maintain a personal handbook for quick reference and revision.

## Topics Covered

- Introduction to Java
- Variables and Data Types
- Literals
- Operators and Expressions
- User Input
- Strings
- Conditional Statements (`if`, `if-else`, `switch`)
- Loops (`for`, `while`, `do-while`)
- Practice Sets

## Objectives

- Strengthen Java programming fundamentals.
- Practice coding through exercises and problem-solving.
- Document important concepts and observations.
- Build a well-organized collection of Java programs for future reference.

## How to Run

Compile the Java file:

```bash
javac FileName.java
```

Run the program:

```bash
java FileName
```
