# Design Patterns and Principles

This repository contains my Java implementations of various software design patterns and principles. It was created as part of my learning journey to better understand object-oriented programming and how design patterns help in building clean, reusable, and maintainable applications.

Throughout this project, I focused on writing simple, easy-to-understand code while following the core concepts behind each pattern. Working on these implementations helped me gain hands-on experience with solving common software design problems and improving code organization.

### Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Design Patterns

### What I Learned
- Writing modular and reusable code
- Applying object-oriented design principles
- Understanding real-world use cases of design patterns
- Improving code readability and maintainability
